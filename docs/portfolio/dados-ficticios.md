## Regras para capturas

- Usar somente os nomes, e-mails, telefones e valores deste arquivo ou outros dados igualmente ficticios.
- Nao mostrar barra do navegador com IP, dominio real, porta interna sensivel ou ambiente da cliente.
- Nao abrir DevTools, logs, terminal, banco, cookies, headers, `.env` ou variaveis de ambiente durante a gravacao.
- Remover metadados dos arquivos antes de publicar.
- Revisar cada imagem em zoom antes de adicionar ao README ou ao perfil.
